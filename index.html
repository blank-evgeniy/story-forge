Простые правила — бесконечные вариации сюжета

01

Создайте комнату

Хост задаёт настройки, получает 4-значный код и QR — и делится ими с друзьями одним нажатием.

02

Пишите по очереди

Каждый добавляет одно предложение за 30–90 секунд. В Blind Mode видно только предыдущую фразу — и больше ничего.

03

Читайте вместе

После всех раундов история раскрывается фраза за фразой. Гарантировано: никто не ожидал такого финала.

Возможности

Всё для весёлой игры

Продуманные инструменты, которые не мешают, а помогают

Blind Mode

Каждый видит только последнее предложение. Хаос управляем, результат непредсказуем.

Таймер раунда

30, 60 или 90 секунд на ход. Ограничение времени рождает лучшие идеи.

Вход по QR-коду

Наведи камеру — и ты уже в игре. Никаких регистраций и паролей.

Реальное время

Всё происходит мгновенно для всех участников.

Несколько историй

Каждый игрок строит свою ветку. В финале историй столько же, сколько игроков.

Для любой компании

Вечеринки, тимбилдинг, онлайн-посиделки. Достаточно двух человек, чтобы начать.
