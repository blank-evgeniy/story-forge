# shared

Здесь хранится всё, что не привязано к конкретной странице или сущности.

В отличие от `/entities`, код в `/shared` не несёт доменной логики — только универсальные инструменты, которые можно использовать в любой части приложения.

**Структура:**

```
shared/
  ui/       — базовые компоненты без доменной логики (Button, Input, Dialog и др.)
  api/      — клиент HTTP-запросов и WebSocket-подключения (исключение: здесь хранятся конкретные запросы с доменной логикой)
  hooks/    — универсальные React-хуки
  lib/      — вспомогательные утилиты (конфигурация, i18n, тестовые утилиты)
  icons/    — svg иконки
  types/    — общие TypeScript-типы
```
