# entities

Здесь хранится только повторно используемая доменная логика.

По умолчанию вся доменная логика живёт прямо во `/views` — приложение не предполагает сильной связанности между страницами. Если логика нужна нескольким страницам, она выносится сюда, разбивается по сущностям и размещается в соответствующей директории.

**Пример:** `player-store` → `/entities/player`
